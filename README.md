# Pràctica de Fonètica PAU

## Definició del projecte
Aquest projecte consisteix en la creació d'una aplicació web interactiva tipus test per a practicar i resoldre els dubtes sobre la fonètica del valencià avaluada a les proves PAU. El recurs està dirigit específicament a l'alumnat de 2n de Batxillerat per tal de treballar la distinció de vocals obertes i tancades, consonants sibilants i palatals (sordes i sonores), i els enllaços fonètics. 

L'abast mínim d'aquesta primera versió serà una aplicació extremadament senzilla, construïda en un únic fitxer `index.html`, que mostrarà una pregunta pràctica basada en normativa oficial, quatre opcions de resposta i un botó de correcció.
