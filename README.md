# Pràctica de Fonètica PAU

Aplicació web interactiva tipus test per a practicar la fonètica del valencià de les proves PAU (2n de Batxillerat).

## Definició del projecte
Aquest projecte consisteix en la creació d'una aplicació web interactiva tipus test per a practicar i resoldre els dubtes sobre la fonètica del valencià avaluada a les proves PAU. El recurs està dirigit específicament a l'alumnat de 2n de Batxillerat per tal de treballar la distinció de vocals obertes i tancades, consonants sordes i sonores, diftongs i hiats i els enllaços fonètics. 

L'abast mínim d'aquesta primera versió serà una aplicació extremadament senzilla, construïda en un únic fitxer `index.html`, que mostrarà una pregunta pràctica basada en normativa oficial, dues opcions de resposta i un botó de correcció.
